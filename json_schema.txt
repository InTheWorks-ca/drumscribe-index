Field rules:

tracks
  Required array of song/chart-family records.

track.id
  Required stable unique id for the song/chart family.
  This id is shared by regular and simplified versions.

track.artist
  Required display artist name.

track.title
  Required display song title.

track.metadata
  Optional shared song metadata.
  Applies to the song, not to a specific chart version.
  If omitted, metadata is unknown or not currently indexed.

track.metadata.year
  Optional release year.
  Use null if present but not assigned or unknown.

track.metadata.album
  Optional album name.
  Use null if present but not assigned or unknown.

track.metadata.genres
  Optional array of genre strings.
  Use an empty array if present but no genres are assigned.

track.metadata.tags
  Optional array of tag strings.
  Use an empty array if present but no tags are assigned.

track.versions
  Required object containing available chart versions.

track.versions.regular
  Required standard chart arrangement.

track.versions.simplified
  Optional simplified chart arrangement.
  Present only when a simplified chart exists.
  Do not create placeholder simplified versions.

version.difficulty
  Required difficulty for that specific chart version.
  Use null if not assigned.

version.videos
  Required for regular versions.
  Optional for simplified versions when no simplified-specific videos exist.
  If present, it must contain playthrough.

videos.playthrough
  Required whenever videos is present.
  Video format where performance video is shown and notation/text appears near the bottom.

videos.scrolling_sheet
  Optional video format where the scrolling sheet occupies the screen.
  Omit this object when no scrolling-sheet videos exist.
  Do not create placeholder scrolling_sheet objects.

videos.scrolling_sheet.light
  Optional light-theme scrolling sheet video links.
  Omit when no light-theme scrolling-sheet videos exist.

videos.scrolling_sheet.dark
  Optional dark-theme scrolling sheet video links.
  Omit when no dark-theme scrolling-sheet videos exist.

cover_url
  Required inside playthrough and inside any present scrolling_sheet theme object.
  Drum performance mixed with the drumless song.
  Use null if unavailable.

drums_only_url
  Required inside playthrough and inside any present scrolling_sheet theme object.
  Only the drum performance audio.
  Use null if unavailable.

drumless_url
  Required inside playthrough and inside any present scrolling_sheet theme object.
  Song/backing audio with drums removed.
  Use null if unavailable.
